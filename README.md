# Job Application Tracker

A clean, interactive job application tracker built with React + Vite. Track applications, interviews, referrals, and more with table and Kanban board views.

## Features
- Add, edit, delete job applications
- Table view & Kanban board view
- Search, filter by status, sort by any field
- Priority levels, follow-up dates, notes
- Data persists in browser localStorage
